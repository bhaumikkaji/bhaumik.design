Practo Ray on iOS

Practo Technologies is a leading Healthcare software company in India. They have a product named Ray which is a web based practice management for Doctors. Ray has companion apps on android and iOS. The Application currently has two main features, Scheduler and Patients Manager. My internship with Practo was to redesign the iOS companion app for the most apt native and natural iOS user experience. My engagement with Practo was of an Product Design Intern where I executed the project in 6 months ( July 2015 - Dec 2015 ).

ROLE AND RESPONSIBILITY

Improve the existing user experience of the application, clean up the existing cluttered user interface with minimalism.

Make Appointments/Calendar features better for on the go users. Make the App interactions in pace with the current iOS version.

Improve the Patient record management with a rich timeline experience.

Project Focus

  • Interaction Design
  • Craftmanship
Full Case Study

IMPACT

The new redesigned app aims to improve the user experience of the existing and new iOS users drastically. It is designed to perform the Practice management operations faster for the on the go use.

Even with the newer design refresh, the fundamentals that I had laid during this project are still integral.
